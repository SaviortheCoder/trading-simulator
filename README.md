# MERN + Next.js Trading Simulator

<!-- Project Description -->
A full-stack stock and crypto trading simulator with real-time price updates.
Users can practice trading without risking real money.

## Tech Stack

<!-- Backend Technologies -->
- **Backend:** Node.js, Express, MongoDB, Socket.io
  - Node.js: JavaScript runtime for server
  - Express: Web framework for building REST APIs
  - MongoDB: NoSQL database for storing user data
  - Socket.io: Real-time bidirectional communication (live prices)

<!-- Frontend Technologies -->
- **Web:** Next.js 14, TypeScript, Tailwind CSS
  - Next.js: React framework with SSR/SSG
  - TypeScript: Type-safe JavaScript
  - Tailwind CSS: Utility-first CSS framework

<!-- Mobile Technologies -->
- **Mobile:** React Native, TypeScript
  - React Native: Build native iOS/Android apps with React
  - TypeScript: Type-safe code across platforms

<!-- Shared Code -->
- **Shared:** TypeScript types and utilities
  - Shared between web and mobile for consistency

## Project Structure

<!-- Folder Organization -->
- `backend/` - Express API + WebSocket server (Node.js backend)
- `web/` - Next.js web application (browser version)
- `mobile/` - React Native mobile app (iOS/Android)
- `shared/` - Shared TypeScript code (types, constants, utilities)

## Features

<!-- Core Features List -->
- User authentication (JWT tokens for secure login)
- Buy/sell stocks and crypto (simulated trading)
- Real-time price updates via WebSocket (live market data)
- Portfolio tracking with P&L (profit/loss calculations)
- Transaction history (view all past trades)
- Watchlist with price alerts (get notified when prices hit targets)
- Robinhood-inspired dark theme (black + green design)
